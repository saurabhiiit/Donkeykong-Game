Terminal based Donkeykong game

language used python

In this game player has to reach to the queen to win the game . player can collect the ramdomly generated coin to increase the score. Player can jump the strair to go to next level but it can't jump broken stair. Donkey at the top left has the random movement and produces the fireball . If player come in contact with fireball then it looses its life. 

Controls-
Key A - Move left
Key D - Move right
Key W - Move up
Key S - Move down
